# Django Boiler Plate

##Enviroment Setup:
```
$ git clone https://www.github.com/hammadarshad1/Django-Boiler-Plate
$ python3 -m venv venv
```

##Installation:
```
$ pip install -r requirements.txt
```

##Usage
```
$ python manage.py migrate
$ python manage.py createsuperuser
$ python manage.py runserver
```
